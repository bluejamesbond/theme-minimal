# theme-minimal

A minimal theme for Jekyll Bootstrap based on [minimal](https://github.com/orderedlist/minimal) by orderedlist.

## Installation

Just run
`rake theme:install git="https://github.com/swaeku/jekyll-theme-minimal.git"`

And you're done.
